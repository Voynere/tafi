# ТАФИ — Медицинские центры и лаборатории

Сайт сети медицинских центров и лабораторий **«ТАФИ-Диагностика»** — [tafimed.ru](https://tafimed.ru/).

Уникальный лабораторный комплекс мирового уровня, более 20 лет специализирующийся в решении вопросов лабораторной диагностики.

## Быстрый старт

### Требования
- PHP 7.4–8.1
- MySQL 5.7+ / MariaDB 10.3+
- nginx или Apache с mod_rewrite
- Composer (опционально)

### Установка

1. **Клонировать репозиторий:**
   ```bash
   git clone https://github.com/Voynere/tafi.git
   cd tafi
   ```

2. **Восстановить базу данных:**
   ```bash
   gunzip tafimed_db.sql.gz
   mysql -u root -p sitemanager < tafimed_db.sql
   ```

3. **Настроить подключение к БД** в файле `bitrix/php_interface/dbconn.php`

4. **Настроить веб-сервер** (корень сайта — папка проекта)

5. **Восстановить права:**
   ```bash
   chmod -R 755 bitrix/
   chmod -R 755 upload/
   ```

### Доступ к админке

```
https://tafimed.ru/bitrix/admin/
```

## Разработка

Проект использует стандартную архитектуру 1С-Битрикс. Основные точки расширения:

- **Шаблон сайта:** `bitrix/templates/aspro_max/`
- **Кастомные компоненты:** `bitrix/components/` (не `bitrix/components/bitrix/`)
- **Локальные модули:** `local/modules/`
- **Обработчики событий:** `bitrix/php_interface/init.php`

---

## Исключения из Git

См. `.gitignore`:
- Ядро Битрикса (`/bitrix/*`), кроме шаблонов и кастомных компонентов
- Медиафайлы (`/upload/*`)
- Кэш, бэкапы, логи
- Бэкап базы данных (`tafimed_db.sql.gz`)
